## How to build

- Require `node`, `yarn`

1. `% git clone git@github.com:pastak/scboloo.git`
2. `% cd scboloo`
3. `% yarn install`
4. `% yarn build`
  - You can `yarn watch` to watch your code to build while developing.
5. Install `__WORK_DIR__/dist` on `chrome://extensions`
